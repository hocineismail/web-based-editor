Task

This project is built using Create React App, Redux, and styled-components, React Bootstrap. It includes a flow diagram to manage sequences and capabilities for a machine.

  • Install Node.js

Features:

  1. Technology Stack:
    • React
    • TypeScript
    • Redux

Use cases to be covered

The editor should be able to do this:

  • Load the MachineCapabilities file
  • Load an existing sequence from a json file like sequence_assessment.json
  • Create a new, empty sequence
  • Save (download) a sequence as a json file
  • Display all process steps of a sequence
  • Add a new step
  • Delete step
  • Select which function is to be executed in a step
  • Edit the successor of a step
  • Select successor step for `Failed
  • Select successor step for `Succeeded To run program, run the following command.

Setup and Usage

  1. Clone the repository: git clone https://github.com/hocineismail/challenge-cities.git

🔨 Running Code

To run the program, execute the following commands:

  1. Run Code Source: Use the terminal to execute the following commands:

    • Navigate to the "task" directory where the frontend code is located using cd task.
    • Install the dependencies for the React application using npm install or yarn install.
    • Start the application using npm start or yarn start.

  2. Open Executable HTML:

    • Navigate to the build folder and open index.html in a web browser.

  3. View Generated Documentation:

    • Navigate to the docs folder and open index.html in a web browser.
